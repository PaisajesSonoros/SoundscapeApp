Crear Cuestionario
Esta función sirve para generar un cuestionario introduciendo archivos de audio y si es necesario, también se puede introducir archivos con apoyo visual.

Una vez generado, se podrá rellenar en la sección "Rellenar Cuestionario".

El sistema de creación es muy sencillo, se selecciona el número de lugares y el número de sistemas de grabación o grabaciones distintas por lugar.

Pulsa en "Más Info" si quieres una mayor explicación.
Rellenar Cuestionario
Después de crear los cuestionarios se pueden estos rellenar por parte de los partipantes. Los cuestionarios que muestra esta aplicación son "Cuestionarios de Paisajes Sonoros".

Un cuestionario sonoro es una lista de preguntas con el fin de evaluar el entorno acústico de un lugar tal y como es percibido, experimentado o entendido por las personas.

Cada cuestionario realizado se irá apilando dentro de la aplicación o en su defecto, se podrá exportar en un JSON.

Pulsa en "Más Info" si quieres una mayor explicación.
Mostrar Resultados
Esta función sirve para comprobar y mostrar los Resultados de un cuestionario creado. Este menú sirve para ayudar al analisis y obtención de conclusiones al encuestador.

En este apartado aparecerán gráficas y tablas con los datos de cada pregunta. En función del número de respuestas, cada apartado será más detallado.

Además, al principio se mostrará un pequeño resumen con la información genérica de los cuestionarios

Pulsa en "Más Info" si quieres una mayor explicación.

¿Quiénes Somos?

Este proyecto fue desarrollado por Manuel Salcedo en la realización del Trabajo Final de Grado: "Evaluación perceptual de paisajes sonoros en remoto a través de una aplicación" de la titulación Ingeniería en Sistemas Audiovisuales y Multimedia. El tutor de dicho trabajo fue Roberto San Millán Castillo. Fue presentando en el curso académico 2022/2023.

El objetivo del desarrollo de esta aplicación fue permitir la recogida masiva de cuestionarios en remoto, facilitando su recogida.

El desarrollo de este software busca mejorar los servicios ya existentes para recoger los cuestionarios facilitando tanto si se usa de forma remota como si se usa en un laboratorio y con un supervisor.

SoundscapeApp se divide en tres partes principales: Creación del cuestionario, Introducción de datos por parte de los participantes y Muestra de los resultados.

¿Qué opciones extra tiene la aplicación?

Opción administrador

Con esta opción se puede esconder algunas opciones que a la hora de realizar una recogida masiva de datos como por ejemplo el crear un cuestionario o mostrar resultados.

Unir cuestionarios

Si se realizan muchos cuestionarios sueltos de un mismo tipo estos se pueden unir todos en un mismo fichero. De esta forma se reduce la cantidad de archivos y se evita posibles pérdidas.

Ayuda

Dentro de la opción "Ayuda" se podrá ver una guía con información adicional de cada elemento relevante. El objetivo es servir de referencia para poder realizar correctamente cada paso.

Resetear Aplicación

Puede darse el caso de que se desee eliminar la información que aparece en la aplicación. Para ello, existe esta opción. Una vez usada, se formatea la aplicación.

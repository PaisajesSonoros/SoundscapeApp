Create Questionnaire
This function is used to generate a questionnaire by introducing audio files and if necessary, you can also enter files with visual support.

Once generated, it can be filled in the "Fill Questionnaire" section.

The creation system is very simple, you select the number of places and the number of recording systems or different recordings per location.

Click on "More Info" if you want further explanation.
Fill Questionnaire
After creating the questionnaires, they can be filled by the participants. The questionnaires that this application shows are "Soundscape Questionnaires".

A soundscape questionnaire has a list of questions in order to evaluate the acoustic environment of a place as it is perceived, experienced or understood by people.

Each questionnaire completed will be stacked within the application or, failing that, it can be exported in a JSON.

Click on "More Info" if you want further explanation.
Show Data
This function is used to check and display the Results of a created questionnaire. This menu serves to help the interviewer analyze and obtain conclusions.

In this section, graphs and tables will appear with the data for each question. Depending on the number of responses, each section will be more detailed.

In addition, at the beginning a small summary will be shown with the generic information of the questionnaires.

Click on "More Info" if you want further explanation.

About us

This project was developed by Manuel Salcedo in the completion of his Final Degree Project: "Perceptual evaluation of remote soundscapes through an application" of the degree Audiovisual and Multimedia Systems Engineering. The tutor of this project was Roberto San Millán Castillo. It was presented in the 2022/2023 academic year.

The objective of developing this application was to allow the mass collection of questionnaires remotely, facilitating their collection.

The development of this software seeks to improve existing services to collect the questionnaires facilitating whether used remotely or used in a laboratory and with a supervisor.

SoundscapeApp is divided into three main parts: Create Questionnaire, Fill data by the participants and Show results.

Extra Options

Administrator Option

With this option you can hide some options when carrying out massive data collection, such as creating a questionnaire or showing results.

Join Questionnaires

If many individual quizs of the same type are carried out, they can all be united in the same file. In this way, the number of files is reduced and possible losses are avoided.

Help/Support

Within the "Help" option you can see a guide with additional information on each relevant element. The objective is to serve as a reference to be able to correctly carry out each step.

Format App

It may be the case that you want to delete the information that appears in the application. For this, there is this option. Once used, the application is formatted.
